# Sub-Component 7.4: Analysis Viewer

## Purpose
View storyline analysis results — timeline, characters, themes.

---

## Requirements Gathering

### Question 1: Timeline View

**Timeline view — linear, branching, or both?**

Options:
- **A. Linear:** Simple chronological list
- **B. Branching:** Visual tree for non-linear stories
- **C. Both:** Toggle between views
- **D. Interactive:** Zoomable, pannable timeline
- **E. Condensed:** Summary view with expand details

### Question 2: Character Display

**Character display — grid, list, or relationship graph?**

Options:
- **A. Grid:** Visual cards with portraits
- **B. List:** Text-focused, sortable
- **C. Relationship graph:** Network visualization
- **D. Tabs:** Different views accessible
- **E. Interactive profile:** Click for full details

### Question 3: Analysis Status

**Analysis status — inline or separate progress view?**

Options:
- **A. Inline:** Progress shown in viewer
- **B. Separate:** Dedicated progress page
- **C. Modal:** Popup during analysis
- **D. Notification:** Toast updates
- **E. Background:** Silent with completion alert

### Question 4: User Manipulation

**How much can users manipulate the analysis?**

Options:
- **A. Read-only:** Purely viewing
- **B. Corrections:** Fix errors in analysis
- **C. Annotations:** Add notes and highlights
- **D. Reordering:** Rearrange timeline
- **E. Full editing:** Complete modification

### Question 5: Theme Exploration

**How should themes be displayed?**

Options:
- **A. Tag cloud:** Visual word cloud
- **B. List:** Ranked theme list
- **C. Timeline overlay:** Themes on timeline
- **D. Color-coded:** Visual theme mapping
- **E. Relationship map:** Theme connections

### Question 6: Detail Level

**Detail presentation — inline expand or modal?**

Options:
- **A. Inline expand:** Expand in place
- **B. Modal:** Popup for details
- **C. Sidebar:** Slide-out panel
- **D. New page:** Full page for details
- **E. Tooltip:** Hover for quick info

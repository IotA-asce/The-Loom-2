**What can users modify in branches?**

Options:
- **A. Nothing:** View only

### Question 4: Selection Workflow

**Selection — single or multiple branches?**

Options:

- **B. Multiple:** Save several for later

Options:
- **A. Title only:** Just branch name
- **B. Summary:** Brief description
- **C. Detailed outline:** Chapter breakdown

- **E. Interactive:** Explore before committing

**Export from branch studio?**

Options:
- **A. No export:** Must go to reader
- **B. Branch only:** Export premise/outlines
